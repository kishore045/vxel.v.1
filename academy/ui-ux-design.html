1. Which part helps start the fridge compressor?

Answer: Relay

2. No cooling complaint can be related to which part?

Answer: Compressor

3. Door cooling loss can happen because of?

Answer: Weak door gasket

4. Thermostat mainly controls what?

Answer: Temperature

5. Before repair, technician should confirm?

Answer: Power supply and safety
